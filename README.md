# amazon-clone

Amazon UI Clone Project built with HTML, CSS, and JavaScript, using Local Storage.

## Live Project Link
[View Live Project](https://khatravathpriyanka2705-gif.github.io/amazon-clone/)

## Project Features
- Product listing with images, names, and prices
- Add to Cart functionality
- Cart stored in Local Storage
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage
